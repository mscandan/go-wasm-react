# GoWasmReact

## Getting Started

- You need to have Go installed on your system

  ```sh
      brew install go
  ```

- You need to have NodeJS installed on your system

  ```sh
      brew install node
  ```

- Install dependencies

  ```sh
      go mod download && npm install
  ```

- Run on dev-mode (no hot-reload)

  ```sh
      npm run serve:fe
  ```

- Build app

  ```sh
      npm run build:all
  ```
